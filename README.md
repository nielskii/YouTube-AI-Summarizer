# YouTube-AI-Summarizer
Ferramenta que automatiza a extração de conteúdo do YouTube. O projeto utiliza Pytubefix para download, Whisper para transcrição local e a API da Groq (Llama 3) para gerar resumos inteligentes. Desenvolvido com Flask, transforma horas de audiovisual em textos concisos (.txt) de forma rápida, unindo IA e automação de forma eficiente.
